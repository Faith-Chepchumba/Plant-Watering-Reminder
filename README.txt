# Plant Watering Reminder

## Overview

A web-based application that reminds users to water their plants after a specified number of days.

## Features

- Enter plant name and watering frequency.
- Displays a reminder message.

## Technologies Used

- HTML
- JavaScript

## How to Run

1. Open `plant_reminder.html` in a web browser.

2. Enter your plant name and watering frequency, then click "Set Reminder".
